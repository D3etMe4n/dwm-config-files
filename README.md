# DWM Config

### Include slstatus, dmenu, st

#### Require: 

* **Feh** for backgrounds
* **Ranger** as file manager
* **flameshot** - for screenshots
* **XF86keysym**(include), **brightnessctl** and **pulseaudio(pactl)** for the volume and brightness control
* **Ubuntu Mono Nerd Font** - for fonts

#### DWM Patches:
* **alwayscenter**
* **attachbottom**
* **autostart**
* **fibonacci**
* **pertag**
* **preserveonrestart**
* **restartsig**
